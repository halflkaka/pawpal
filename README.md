# Pet Health

A small personal-use pet health app project.

## Structure

- `PawPal/` — iOS SwiftUI app
- `pet-health-backend/` — FastAPI backend
- `PawPal.xcodeproj/` — Xcode project

## Current goal

Build a minimal personal pet health app that can:
- enter pet symptoms
- get simple analysis
- find nearby vets
- store local history
